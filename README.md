## 사이트 설명
충청남도 금산의 버스 정보를 담고 있는 사이트입니다.

국토부 api에 등록되어 있는 금산 버스 데이터의 내용이 적어 금산 군청 사이트의 등록된 엑셀 파일을 json으로 만들어 사용하였습니다.

데이터로 가져오는 것이 아니라 유지보수의 힘듦이 있겠지만 고향의 버스 정보를 담은 앱이 없어서 만들었습니다.
